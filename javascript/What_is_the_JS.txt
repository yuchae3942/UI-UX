프로그래밍 기초
- 컴퓨터에게 우리가 해야 될 일을 시키는 명령문을 순서대로 작성
- 해당하는 일을 하기 위한 모든 과정을 순서대로 작성
    예)
    숫자 두개를 입력 받아서 두 수의 합을 출력하는 코드를 작성
        1. 숫자입력
        2. 숫자입력
        3. 합을 구함
        4, 결과를 출력

    
    1. 데이터 저장할 공간 - 메모리 컨트롤
    2. 입력 / 출력
    3. 연산자 - 사칙, 복합대입, 증감, 비교, 논리 
    4. 제어문 - 분기(경우의 수?), 반복
    5. 함수



javascript
인터프리터형 언어: 사용자가 작성한 코드를 한줄씩 읽어서 해석하여 실행
<script> 태그 안에 작성


windows, macintosh, iphone, android, linux

    사용자가 웹페이지에서 하는 행동
    onclick
